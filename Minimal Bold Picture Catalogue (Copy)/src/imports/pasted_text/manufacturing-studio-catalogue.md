# Visual Style

I would avoid putting long bullet lists everywhere.

Instead convert every service into one of these:

•⁠  ⁠Large process photo
•⁠  ⁠Material swatch
•⁠  ⁠Manufacturing icon
•⁠  ⁠CAD screenshot
•⁠  ⁠Texture macro
•⁠  ⁠Workshop photography
•⁠  ⁠Technical drawing
•⁠  ⁠Exploded assembly
•⁠  ⁠Timeline
•⁠  ⁠Flowchart

The ratio should be roughly:

*70% imagery*

*20% diagrams*

*10% text*

---

# Information Hierarchy

Rather than listing everything in one place, distribute the information naturally.

| Section          | What to show                            |
| ---------------- | --------------------------------------- |
| Philosophy       | Vision and approach                     |
| Services         | Overview of capabilities                |
| Design           | Research, strategy, concept development |
| Prototyping      | 3D printing and model making            |
| Metal            | Materials, fabrication, detailing       |
| Surface Finishes | Finish samples and coatings             |
| Wood             | Fabrication and finishing               |
| Leather          | Soft goods development                  |
| Artisan Work     | Decorative craftsmanship                |
| Production       | Manufacturing support and quality       |
| Projects         | Real applications                       |
| Clients          | Industries served                       |
| Contact          | Call to action                          |

This structure reads like a premium studio catalogue rather than a service brochure, and mirrors the architecture used by leading manufacturing and design firms: *philosophy → capabilities → materials → processes → craftsmanship → projects → collaborators → contact*. It will also give you plenty of opportunities for full-bleed visuals, close-up material photography, and clean editorial layouts that feel high-end rather than corporate.
